# Инструкция по установке и запуску

## Требования

- Node.js 18+ 
- npm или yarn

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

### Режим разработки
```bash
npm run dev
```
Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка для продакшена
```bash
npm run build
npm start
```

## Использование

1. **Регистрация**: Нажмите "Sign Up" и заполните форму
2. **Вход**: Используйте email и пароль для входа
3. **Поиск пар**: Перейдите в раздел "Matches" и используйте фильтры
4. **Сообщения**: В разделе "Messages" вы можете общаться с другими пользователями
5. **Профиль**: В разделе "Profile" просматривайте и редактируйте свой профиль

## Особенности

- Все данные сохраняются в localStorage браузера
- При перезагрузке страницы вы остаетесь авторизованным
- Фильтры в разделе "Matches" работают в реальном времени
- Сообщения можно отправлять и получать
- Секция контактов в футере теперь работает корректно

## Структура данных

Данные пользователей хранятся в localStorage:
- `datingAppUser` - текущий авторизованный пользователь
- `datingAppUsers` - список всех зарегистрированных пользователей

## Решение проблем

Если возникают ошибки:
1. Убедитесь, что установлены все зависимости: `npm install`
2. Очистите кэш: `rm -rf .next node_modules && npm install`
3. Проверьте версию Node.js: `node --version` (должна быть 18+)
